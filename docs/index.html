已包含的 Mod

下载 Release 压缩包后，直接把顶层 `mods` 文件夹复制到游戏目录即可。

多人伤害统计

多人模式伤害 HUD 与战斗摘要，区分之前战斗累计伤害和当前战斗伤害。

游戏速度调节

暂停菜单速度切换，支持 1x 到 4x；多人模式中以房主速度为准。

暂停菜单重打

单人模式暂停菜单快速重打，避免影响多人同步流程。

中文调试控制台

按 F10 呼出中文覆盖层，支持添加/移除卡牌和遗物。

观者角色

为《杀戮尖塔 2》加入一代观者角色，包含 DLL 逻辑和 PCK 资源。

Card Effect Tweaks

卡牌或效果调整类 Harmony Mod 模板，适合继续扩展新玩法。

安装方式

  1. 点击页面顶部“下载最新版”会直接下载 `STS2mods-v1.0.16.zip`。
  2. 解压压缩包，把里面的 `mods` 文件夹复制到《杀戮尖塔 2》游戏目录。
  3. 启动游戏，在 Mod loader 中确认对应 Mod 已加载。
建议

安装或更新 Mod 前先备份存档。如果需要同步官方档和 Modded 档的 `progress.save`，可以配合使用 STS2-Save-Sync-Tool。
